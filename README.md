# PyTorch-YOLOv3
Forked from BoLiu's Pytorch-YOLOv3
Full implementation of YOLOv3 in PyTorch.

required:
    pytorch: 0.4.0
    anaconda: 3.6.0

    bash get_coco_dataset.sh

    You need to rectify the directory(cause that I used many absolute path) in the test_images.py
